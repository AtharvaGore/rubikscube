# RUBIKS CUBE

### USAGE INSTRUCTIONS FOR LINUX

<ul>
<li> clone the repository
<li> cd to the folder and run "make"
<li> run the executable file "rubiks.out"
</ul>

### CONTROLS

R => Rotate right layer
L => Rotate left layer
U => Rotate upper layer
Shift + R or L or U => Rotate in the opposite direction
Arrow keys => Rotate the entire cube
